# Capturing user input
```go
import (
    "bufio"
    "os"
)
```
<table>
  <thead>
    <tr>
      <th>Step</th>
      <th>Code</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1. Create scanner</td>
      <td><code>scanner := bufio.NewScanner(os.Stdin)</code></td>
      <td>Points the scanner at standard input</td>
    </tr>
    <tr>
      <td>2. Block for input</td>
      <td><code>scanner.Scan()</code></td>
      <td>Blocks and waits until the user presses enter</td>
    </tr>
    <tr>
      <td>3. Read the text</td>
      <td><code>scanner.Text()</code></td>
      <td>Returns the line the user typed as a string</td>
    </tr>
  </tbody>
</table>
